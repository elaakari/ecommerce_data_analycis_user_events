# End-to-End E-commerce Analytics with SQL & Power BI

This project presents an end-to-end e-commerce data analysis using **SQL**, **SSMS**, and **Power BI** based on the public **TheLook E-commerce** dataset.

The objective is to analyze customer behavior across the purchasing journey, identify conversion bottlenecks, evaluate marketing channel performance, and uncover revenue-driving insights.

## Key Analyses

* **Sales Funnel Analysis** – Track user progression from homepage visit to purchase and identify drop-off points.
* **Marketing Channel Performance** – Compare traffic sources based on conversion rates and revenue contribution.
* **Time-to-Conversion Analysis** – Measure the average time between a user's first visit and their first purchase.
* **Revenue Analysis** – Analyze revenue trends, Average Order Value (AOV), and channel profitability.
